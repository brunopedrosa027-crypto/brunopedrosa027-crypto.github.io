Sobre Mim

Da operação à análise: transformando processos através dos dados

Minha jornada com dados começou na prática, identificando gaps em processos reais de logística de veículos. Como auxiliar administrativo, percebi que as decisões eram tomadas sem o suporte de dados consolidados.

Em apenas 2 meses, evolui de planilhas manuais no Excel para desenvolver dashboards completos em Power BI, dominando medidas DAX, modelagem de dados no Power Query e criando pipelines ETL que extraem dados diretamente de APIs REST.

1

Excel e Processos Manuais

Iniciei com planilhas manuais e consultas pesadas no Excel, identificando a necessidade urgente de automação.

2

Automação com Power Query

Desenvolvi ETLs automatizados extraindo dados de APIs REST, reduzindo o tempo de processamento em 70%.

3

SQL e Arquitetura Escalável

Migrei para SQL Server (Azure), criando views otimizadas que melhoraram a performance em 5x.
