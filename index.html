Sobre Mim

Nos últimos meses, iniciei uma jornada intensa no mundo de dados. Comecei criando dashboards no Excel e, rapidamente, evoluí para trabalhar com JSONs extraídos de APIs REST, tratá-los no Power Query e construir visualizações úteis.

Atualmente curso o 1º período de Análise e Desenvolvimento de Sistemas e venho implementando consultas em Azure SQL para estruturar os dados antes da modelagem no Power BI — melhorando performance e organização. Ainda estou em evolução, mas já desenvolvi dashboards operacionais que ajudam a tomar decisões no dia a dia.
